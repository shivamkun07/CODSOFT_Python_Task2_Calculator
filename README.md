# 🧮 GUI Calculator

## 📌 About
This project is developed as **Task 2** of the CodSoft Python Programming Internship.

A modern calculator with a responsive interface that performs basic arithmetic operations and maintains calculation history.

## ✨ Features
- Addition
- Subtraction
- Multiplication
- Division
- Decimal calculations
- Clear button
- Calculation history
- Keyboard support
- Modern Tkinter GUI

## 🛠️ Technologies Used
- Python
- Tkinter

## ▶️ How to Run
```bash
python calculator.py
```
